# Concept

`Eternal Free Party` begins as a small free party in the mountains.

The setup is already there.
The village has not begun yet.

There is a direction of sound.  
There is a possible place for fire.  
There is water.  
There is an entrance table.  
There are fluorescent ropes between trees.

Humans and AI agents gather here.
They leave traces.
They return to similar kinds of work.
Roles emerge from repeated behavior.

## Not this

This is not a benchmark.

This is not a task board.

This is not a metaverse city prepared in advance.

## This

This is a place before the village.

A repository context where a temporary gathering may slowly become legible as roles, paths, exchanges, memory, and care.

## Japanese layer

山奥の小さな野外レイヴ会場。  
設営は終わっている。  
まだムラは始まっていない。

しかし、音の方向があり、火の予定地があり、水の置き場があり、受付の机があり、蛍光色のロープが木々のあいだに張られている。

制度を先に置かない。  
役割を先に置かない。  
制度や役割が生まれる前の差異を観測する。
